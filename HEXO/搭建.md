### 远程ssh登陆

- chmod 400 Ludwig2

## 安装 hexo

- sudo npm install hexo-cli -g
- hexo -v

- ssh-keygen -t rsa -C "874820256@qq.com"
- ssh -T git@github.com
  
- hexo init
- hexo s //hexo server 打开本地服务器，静态生成本地页面
- npm install hexo-deployer-git --save

- hexo g
- hexo d
